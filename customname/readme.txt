== Description ==

Customname takes your name as an input. After entering any name in it, this plugin returns your name in your favorite color and also returns your lucky flower.

Major features in Customname includes:

* Automatically display name in your favorite color.
* Validation is done for each input field.


== Installation ==

Upload the Customname plugin to your plugin folder of site, activate it on admin plugin page , and then at the end of each and every page you can access this plugin.
